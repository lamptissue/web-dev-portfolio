Film Finder

Film Finder allows users to explore and discover a range of films. By using a custom-built API, users can access detailed information about the films such as release dates, director and genre. Users can also create their own list of favourite films. The website is developed using Node.js and Express.js for the backend, MongoDB for data storage, React for the frontend and Styled Components for the CSS.

Objective

The objective of this project is to gain a good understanding of building a custom API and also a deeper understanding of React and React features such as Hooks which is a useful feature for managing state and enabling more efficient functional components.

Approach

The project began by assembling a collection of 12 films from the top 20 films of all time, storing the details in a JSON file including film summaries, release dates, directors, and director biographies. To ensure proper connectivity, I created a simple API with GET requests using Postman to test fetching all films, specific films, films by director, and films by genre. Once verified locally, I integrated MongoDB using Mongoose schema for data storage. To make sure everything was working correctly, I created a simple website using JavaScript and a fetch request to retrieve film data from the API.

Moving onto front-end development, I opted for React due to its popularity, component-based architecture, and efficient rendering capabilities. Starting with the film grid, I built a dynamic layout to render the films. Then, I focused on creating a side panel displaying more detailed information on the films, and then a film search functionality. Lastly, I implemented a favourite feature, enabling users to select their preferred films and easily access them with a click of a button, enhancing interactivity and user experience on the Film Finder website.

Key learnings and next steps

Throughout the project, I gained a valuable understanding of React Hooks and their significance in managing state and building more efficient components. Looking ahead, the next steps involve expanding the website's functionality by implementing user authentication, enabling users to create profiles, and allowing them to save their favourite films. Lastly, adding POST, PUT, and DELETE requests to the API will allow users the ability to create, update, and delete favourite films and their profiles, further enhancing the functionality and interactivity of the Film Finder website.
